Local storage is a feature that allows websites to store data on a user's device: JavaScript A built-in feature of web browsers that allows developers to store data in the user's browser. Local storage is a reliable solution for small amounts of data, like key-value assignments. It's a good tool for applications that need to save user preferences or data locally, such as a to-do list application. Note : both key and value must be string , we can you json method to store object in the local storage
